JSA JavaScript Accordion

Developed by Homero Cavazos

The accordion is based on the Description List element.

Download
Markup

To activate the accordion add .jsa class to the <dl> tag. The dt needs to have an anchor tag for best accessibility experience. See JavaScript section for settings.

<dl class="jsa">
  <dt><a href="#">Hello World</a></dt>
  <dd></dd>
</dl>
JavaScript

Basic Setup

  var example = new jsa();
  example.init();

Custom Selector

  example.init('#myAccordion');
CSS

The default theme defines the collapse state, even & odd and the toggle open close icons.
